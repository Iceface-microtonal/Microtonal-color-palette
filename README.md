# Microtonal-color-palette
Map microtonal scales to color palettes. I(D,N) builder, .scl import, CSS/JSON export. Tuning is the new skin.

A visual design tool that converts microtonal tuning systems into color palettes. Supports N-EDO, Just Intonation, Maqam, and Iceface I(D,N) tuning with interactive D/N parameters. Exports .scl (Scala), CSS custom properties, and JSON. Built for composers and developers who think in both frequencies and hex codes.
